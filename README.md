# flattenjson

flattenJson.js
-> Flattens a json from A to B

unflattenJson.js
-> Unflattens a json from B to A


A = [{
      "this": "example",
      "is": "example"
    },{
      "an":"example",
      "example":"example"
    }]
B = [{"0.this":"example","0.is":"example","1.an":"example","1.example":"example"}]
